# Basic React Webpack Setup

The most basic webpack implementation to get up and running with React.

### Dependencies

- @babel/core
- @babel/preset-env
- @babel/preset-react
- babel-loader
- css-loader
- html-webpack-plugin
- node-sass
- react
- react-dom
- sass-loader
- style-loader
- webpack
- webpack-cli
- webpack-dev-server
